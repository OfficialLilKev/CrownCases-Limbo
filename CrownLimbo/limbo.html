How it works:
Each roll uses HMAC-SHA256(serverSeed, clientSeed + ":" + nonce) to generate a provably random float u ∈ [0,1). This is converted to a multiplier using the 5% house edge formula: multiplier = 0.95 / (1 - u). You win if the multiplier ≥ your target. P(win) = 95% / target. The server seed is committed (hashed) before the bet and revealed after, so neither party can manipulate results.
